A Cryptographically Secure and Signed Camera App with Public Verification

This app is designed to give authenticity to any Photo, Video, and Audio taken with the App, by sending media data to a secure server where a watermark is burned into it which is then hashed and signed and posted to a public database (blockchain is in development) with a visual code that allows any viewer to easily trace and check authenticity.
